Welcome to Open Graph Image Generator. This plugin enables you to generate Open Graph images automatically with an image overlay of your choice. Additional functionality includes the possiblity of adding the author as well as title with description (using Open Graph tags, if they are present).

== Description ==

Open Graph protocol enables you to make your links more appealing on social media. With usage of this plugin it's also customizable in the way you wish it to be. Choose and customize any image to be embedded in the og:image tag. This plugin currently enables user to include title, description, author as well as choose the background image and overlay image and it's position.

== Frequently Asked Questions ==

= Is it required to install SEO plugins for this plugin to work? =

It is not required, but highly recommended. SEO plugins take care of search engine optimization actions and also generate the tags required for your posts to be more visible on social media and in the search results.

= What's the required resolution for the background image? =

It's 1200px x 630px.
